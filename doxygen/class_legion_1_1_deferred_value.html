Detailed Description

template<typename T>
class Legion::DeferredValue< T >

A deferred value is a special helper class for handling return values for tasks that do asynchronous operations (e.g. GPU kernel launches), but we don't want to wait for the asynchronous operations to be returned. This object should be returned directly as the result of a Legion task, but its value will not be read until all of the "effects" of the task are done. It is important that this object be returned from the task in order to ensure its memory is cleaned up. Not returning any created DeferredValue objects will result in a memory leak. It supports the following methods during task execution:

  • T read(void) const
  • void write(T val) const
  • T* ptr(void) const
  • T& operator(void) const
